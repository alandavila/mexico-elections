Data table with votes per party coallision for each state

2012 presidential election vote tally

This table contains the number of votes that each party coallision obtained in each of Mexico's states. Note that voters were allowed to cast their vote for a number of options in order to support the same candidate. This is due to Mexico's electoral law. If many parties form a coallision, different combinations of their logos might appear in the ballot. The candidate of the coallision would get the sum of all the votes casted in favor of any of such combinations. We will only list the main coalition party in each plot for this analysis.
